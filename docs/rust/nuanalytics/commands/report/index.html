Report generation utilities for CLI commands.

This module provides shared report generation functionality used by multiple CLI commands. It handles loading curriculum data, computing metrics, scheduling terms, and rendering reports in various formats (Markdown, HTML, PDF).

The main entry point is generate_report_file, which orchestrates the full report generation pipeline from an input CSV file.
